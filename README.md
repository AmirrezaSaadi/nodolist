# NodoList

A lightweight Android app for notes and to‑do lists, built for personal daily use.  
Combines a clean interface with essential productivity features.

## ✨ Features

- **Notes + To‑Do Lists** – Create, edit, and delete both notes and tasks.
- **Dark Mode** – Seamless light/dark theme that follows your system preference.
- **Search** – Quickly find any note or task by title or content.
- **Progress Bar** – Visual tracking of your to‑do completion.
- **Smooth Animations** – Polished transitions for a modern feel.
- **Local Storage** – All data saved locally using Room database (no internet required).

## 🛠️ Tech Stack

- **Language** – Kotlin
- **UI Toolkit** – Jetpack Compose
- **Database** – Room (with SQLite)
- **Architecture** – MVVM (ViewModel + LiveData/StateFlow)
- **Other** – Material 3 Design, Coroutines

## 🚀 Getting Started

### Prerequisites

- Android Studio (latest version recommended)
- Android device or emulator (minimum API 21 – Android 5.0)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/AmirrezaSaadi/nodolist.git
2. **Open in Android Studio**

   Select `File > Open` and choose the cloned folder.

3. **Build and run**

   - Click **Run** ▶️ or use `Shift + F10`.
   - No additional setup (API keys, signing configs) is required.

## 📦 Build & Run

The project should compile and run directly after cloning.  
If you encounter issues, try:

- **Sync Project with Gradle Files** (`File` → `Sync Project with Gradle Files`)
- **Clean & Rebuild** (`Build` → `Clean Project`, then `Build` → `Rebuild Project`)
